# PlasmaLabMSU

Репозиторий для студентов и сотрудников лаборатории физики плазмы и физических основ микротехнологии, физического факультета МГУ и отдела микроэлектроники НИИЯФ МГУ.

Большую часть репозитория составляют программы на Labview для управления оборудованием и автоматизацией эксперимента в области низкотемпературной плазмы. Спектроскопия, зондовые измерения. Программы для управления переферийными устройствами: генераторами, расходомерами и т.д.
